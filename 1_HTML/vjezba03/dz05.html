Osnove CSS rasporeda

Tri ključne tehnike: Flexbox, Grid i Float

 Flexbox  Grid  i Float
1.Fleksibilni model kutija (Flexbox) Flexbox je jednodimenzionalni model rasporeda, dizajniran za distribuciju prostora između stavki u spremniku i moćnu mogućnost poravnavanje. Idealan je za navigaciju, male komponente i centriranje elemenata. stavka 1 stavka 2 stavka 3 Ovdje je prikazano poravnavanje i automatska prilagodba veličine stavki unutar reda. 2.Dvodimenzionalni model mreže (CSS Grid) CSS Grid je dvodimenzinalni sustav rasporeda (redovi i stupci). Savršen je za dizajniranje cjelokupnog rasporeda stranice (main layout) gdje želite precizno pozicioniranje elemenata. Zaglavlje Nav. Sadržaj Podnožje Elemnti su smješteni pomoću svojstva grid-template-areas. 3.Klasični raspored (Float & Columns) Prije Flexboxa i Grida, koristio se Float za bočno pozicioniranje elemenata. **Multi-column** omogućuje pakiranje sadžaja u stupce. Float lijevo Ovaj tekst okružuje float element s lijeve strane. Klasično pozicioniranje pomoću float-a pomiče elemente ulijevo ili udesno dok dopušta da se okolni tekst omota oko njega. Također, ovaj odlomak je podijeljen u tri stupca pomoću svojstva **columns**, što je korisno za novinske ili tiskane rasporede. Dodatne informacije Sva tri modela su i dalje relevantna. Flexbox za komponente. Grid za cjelokupni raspored stranice, a Columns za tekstualni blokove. Primjer kreiran za demonstraciju CSS layout svojstava.
